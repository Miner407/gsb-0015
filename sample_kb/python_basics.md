# Python 基础入门

## 变量和数据类型

Python 是一种动态类型语言，不需要声明变量类型。常见的数据类型包括：
- 整数 (int)
- 浮点数 (float)
- 字符串 (str)
- 布尔值 (bool)
- 列表 (list)
- 字典 (dict)

## 函数定义

函数使用 `def` 关键字定义：

```python
def greet(name):
    return f"Hello, {name}!"
```

函数可以有默认参数、可变参数等。函数是 Python 编程的基本构建块。

## 列表操作

列表是 Python 中最常用的数据结构之一。支持以下操作：
- append(): 添加元素
- pop(): 删除元素
- sort(): 排序
- slice: 切片操作

列表推导式是 Python 的特色功能，可以简洁地创建新列表。
