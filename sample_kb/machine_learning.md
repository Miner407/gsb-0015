# 机器学习入门指南

## 什么是机器学习

机器学习是人工智能的一个分支，它使计算机能够从数据中学习并改进性能，而无需进行明确的编程。机器学习的核心是算法和模型。

## 监督学习

监督学习是最常见的机器学习类型。它使用已标记的数据来训练模型，包括：
- 分类问题：预测类别标签
- 回归问题：预测连续值

常见算法包括线性回归、决策树、随机森林、支持向量机等。

## 神经网络基础

神经网络是一种模拟人脑神经元连接方式的计算模型。深度学习使用多层神经网络来处理复杂的模式识别任务。

反向传播算法是训练神经网络的关键技术。

## 常用框架

目前主流的机器学习框架有：
- TensorFlow: 由 Google 开发
- PyTorch: 由 Meta 开发，更易用
- Scikit-learn: 传统机器学习算法库
- Keras: 高级神经网络 API

选择合适的框架可以大大提高开发效率。
